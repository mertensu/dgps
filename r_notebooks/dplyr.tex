The \texttt{tidyverse} is ``a collection of R packages designed for data
analysis'' (see \url{https://www.tidyverse.org/}). Here, we will focus
on one of those packages, namely \texttt{dplyr}. \texttt{dplyr} is ``a
grammar of data manipulation, providing a consistent set of verbs that
help you solve the most common data manipulation challenges'' (see
\url{https://dplyr.tidyverse.org/}). We think that \texttt{dplyr} is
able to (at least in part) replace some basic R commands and make
several tasks a lot easier. Some researchers even think that one should
start teaching \texttt{dplyr} from the very start
(\url{http://varianceexplained.org/r/teach-tidyverse/}). We agree with
this view!

If you encounter any problems, feel free to contact me:\\
{\href{mailto:ulf.mertens@psychologie.uni-heidelberg.de}{\nolinkurl{ulf.mertens@psychologie.uni-heidelberg.de}}}

\texttt{dplyr} mainly consists of six verbs where each verb is optimized
for a specific task.

\begin{itemize}
\tightlist
\item
  \texttt{mutate} adds new variables that are functions of existing
  variables
\item
  \texttt{select} picks variables based on their names.
\item
  \texttt{filter} picks cases based on their values.
\item
  \texttt{summarise} reduces multiple values down to a single summary.
\item
  \texttt{arrange} changes the ordering of the rows.
\item
  \texttt{group\_by} allows to perform any operation ``by group''.
\end{itemize}
